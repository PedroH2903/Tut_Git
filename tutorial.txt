
####### Codigos GitHub ######
git init = inicializar repositório na pasta git
git status  = observar untracked files
git add . = adiciona basicamente todos os arquivos ao commit do git

git comit -a -m  = adicionando ao comit, juntamente com uma mensagem explicar as alterações feitas
git remote add origin https://github.com/PedroH2903/Tut_Git.git
git branch -M main 
git push -u origin main = envia o repositorio atual para o ambiente remoto

git cheackout - b 'teste' = comando para alterar usuário e enviar uma branch
git push --set-upstream origin teste = necessário para crição de branch tanto no respositório local, 
quanto servidor gitub

gitub branch = ver branchs existentes no projeto
git merge teste = unindo com o branch do usuário teste
